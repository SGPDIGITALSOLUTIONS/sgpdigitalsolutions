## Notes
- All pages maintain consistent header and footer
- Mobile responsive design (stacks vertically on smaller screens)
- Clear call-to-action buttons
- Consistent spacing and alignment
- Brand colors to be applied:
  - Deep Blue (`#1a365d`) for primary elements
  - Charcoal (`#2d3748`) for secondary elements
  - White (`#ffffff`) for backgrounds and text
- Navigation menu collapses to hamburger menu on mobile
- All interactive elements should have hover states
- Forms should have proper validation and error states
